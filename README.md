# SignalstoreTutorial

## Packages to install

### [signals](https://ngrx.io/guide/signals) & [ngrx/operators](https://ngrx.io/guide/operators/operators)

````bash
npm install @ngrx/signals --save &&
npm install @ngrx/operators --save
````

## start backend

````bash
npm start-server
````
